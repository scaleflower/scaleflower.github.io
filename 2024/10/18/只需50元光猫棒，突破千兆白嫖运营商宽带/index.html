只需50元光猫棒，突破千兆白嫖运营商宽带

2023-12-03 10:22:19 10点赞 179收藏 14评论

最近都开始普及千兆甚至两千兆宽带了，但是千兆给的都还是千兆猫，除去pppoe等各种损耗，实际上千兆猫只能跑到960M左右的速度，都没达到真千兆_（日内瓦，退钱！）_，实际上千兆宽带可以跑到1200M甚至更高的速度，但是运营商不会为了多这点速度给你更换更好的光猫，不过现在2.5g光猫价格也降了下来，最便宜的的180就可以入手，更是还有只要50的PON Stick ONU（俗称猫棒），运营商不肯换就只能自己花点钱了。

本教程只针对gpon路线！epon速率上限摆在那折腾这个没意义，xgpon等其他类型猫棒偏贵，建议直接上7607p，当然部分线路也是兼容gpon，具体请自测/询问电信小哥。

怎么查看线路类型？拿出你的光猫，查看底部信息，如果有写明gpon onu说明可以用gpon猫棒。

一、准备设备

一个带2.5g口的路由器，如果路由器不带2.5g，2.5g交换机也可以，最好是sfp光口，电口则需要多一个光电转换模块，如果都没有2.5g口那建议不要折腾了，多十几M速度意义不大_（换个2.5g路由器也不贵，两三百）_。

注：不是所有带2.5g sfp的设备都支持猫棒，目前大部分tp、小米等设备都支持，不确定可以购买前询问商家。

一个光猫棒，小黄鱼和淘宝有刷好固件的卖，没刷固件的50元就可以买到。

注：猫棒有几个型号，有华为MA5671A和诺基亚G-010-SA等型号，这里我用诺基亚G-010-SA，因为这个一开始就可以用ssh刷机，比较简单。

二、准备相关宽带信息

  • LOID 及密码 (光猫后台)

  • PASSWORD 密码验证 (光猫后台)

  • 原光猫 SN 序列号 (光猫底部)

  • 原光猫 MAC 物理地址 (光猫底部)

  • 拨号 Vlan 值 (光猫后台)

  • 宽带账户密码（不记得问运营商）

注：光猫后台需要以超级用户登录，不知道账号密码可以百度该型号，文末也附上部分地区相关信息。

提醒：不同运营商认证所需要的信息也不一样，如果上述部分信息没有则说明该运营商不需要此认证信息。

三、刷机

把猫棒接入到路由器/交换机，先把ip段改成192.168.1.X（如果原本就是此ip段就不用动）。

ssh登录猫棒，猫棒默认ip为192.168.1.10，用户名ONTUSER, 密码 SUGAR2A041。

设置环境变量：

fw_setenv asc0 0fw_setenv preboot

重启：

reboot

修改固件名称为sys.image（固件下载地址在文末），上传固件，这里用cmd：

scp 本地固件位置 ONTUSER@192.168.1.10:/tmp

提醒：请确保放入的对应设备型号的固件，否则刷错固件会导致猫棒无法使用！！！

再次登录ssh，查看并记录分区情况，linux和image为系统分区，image分区可能为image0或image1：

cat /proc/mtd图片

进入猫棒目录：

cd /tmp

刷入分区：（请自行修改命令中分区名称，且确保每个分区都刷入，因为一个分区为主分区，另一个为备用分区，当突然断电后有概率切换另一个分区）

mtd -e linux write sys.image linuxmtd -e image0 write sys.image image0

重启：

reboot

四、配置猫棒

固件就是openwrt系统，但去除了拨号等其他功能_（猫棒来拨号，千兆变百兆）_，浏览器输入192.168.1.10进入到猫棒后台，默认账户为root，密码空。

转到gpon设置，填写相关认证信息，保存并应用。

图片

此时返回状态页，gpon注册状态为5，说明已经成功注册，如果不为5，请检查光衰、光纤头污染、以及认证信息是否有误。

图片

转到gpon设置-高级自定义设置，点击切换分区，保存并应用，重启。这时候就会切换到另一个分区，此时还需再配置一遍认证信息。

图片

五、路由器拨号

认证信息配置完成后且gpon注册状态为5，则可以正常拨号，路由器后台输入宽带账户密码拨号即可，此时可以把路由器ip段更换回来。这时候测速就可以轻松突破千兆。

图片

六、其他

1、猫棒发热较大，如果担心散热可以自己diy几个散热片。

2、光衰问题，一般光衰大于-24dBm就可以，但由于猫棒本身是给运营商机房等地方用的，光纤接口是APC，与普通宽带的UPC接口不一样，就导致部分童鞋光衰过大无法认证，这时候加个UPC转APC的转接头即可。

3、猫棒会不会不稳定，现在能买到便宜的猫棒基本上都是二手，都是运营商淘汰的东西，但除去个别猫棒本身问题以及部分版本固件问题，不然和普通光猫稳定性没区别，至少我很少听到说不稳定的情况。

4、猫棒也可以用iptv，vlan这些可以正常使用。
